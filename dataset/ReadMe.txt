There are three files in the folder


1.Genelevel_expectedcounts_matrix.tsv

This file contains the expression levels by raw expected counts (output by RSEM) for 453 individuals that passed quality controls


2. MetaSheet.csv
Metadata for all 453 samples. Variables include age, sex, AMD progression (MGS1/non-diseased to MGS4/advanced AMD).

3.hs_ensembl_85_genelevel_annotations_with_genelengths
This is the annotation file to link the ensemble ids in expression matrix to actual genes name.
